node create

ockam node create [OPTIONS] [NODE_NAME]

Create a node

Arguments:

  • <NODE_NAME>
    Name of the node (Optional)

Options:

  • -f, --foreground
    Run the node in foreground

  • -e, --exit-on-eof
    Watch stdin for EOF

  • -t, --tcp-listener-address <SOCKET_ADDRESS>
    TCP listener address

  • --child-process
    ockam_command started a child process to run this node in foreground

  • --enrollment-token <ENROLLMENT_TOKEN>
    An enrollment token to allow this node to enroll into a project

  • --launch-config <LAUNCH_CONFIG>
    JSON config to setup a foreground node

  • --trusted-identities <TRUSTED_IDENTITIES>

  • --trusted-identities-file <TRUSTED_IDENTITIES_FILE>

  • --reload-from-trusted-identities-file <RELOAD_FROM_TRUSTED_IDENTITIES_FILE>

  • --project <PROJECT>

  • --vault <VAULT>

  • --identity <IDENTITY>

node delete

ockam node delete [OPTIONS] [NODE_NAME]

Delete a node

Arguments:

  • <NODE_NAME>
    Name of the node

Options:

  • -a, --all
    Terminate all node processes and delete all node configurations

  • -f, --force
    Terminate node process(es) immediately (uses SIGKILL instead of SIGTERM)
